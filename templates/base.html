🔒 Kebijakan Privasi

Terakhir diperbarui: 1 Januari 2025

1. Informasi yang Kami Kumpulkan

Cinevu hanya mengumpulkan data minimal yang diperlukan untuk operasional layanan, termasuk data teknis seperti alamat IP, browser, dan perangkat yang digunakan untuk mengakses situs.

2. Penggunaan Data

Data yang dikumpulkan digunakan untuk meningkatkan performa situs, menganalisis trafik, dan menampilkan iklan yang relevan melalui jaringan pihak ketiga (seperti Google AdSense).

3. Cookie

Kami menggunakan cookie untuk menyimpan preferensi pengguna dan data sesi. Kamu dapat menonaktifkan cookie melalui pengaturan browser, namun beberapa fitur situs mungkin tidak berfungsi optimal.

4. Pihak Ketiga

Cinevu menggunakan layanan pihak ketiga seperti Vercel (hosting), Supabase (database), dan jaringan iklan. Masing-masing pihak memiliki kebijakan privasi sendiri.

5. Keamanan

Kami menerapkan langkah-langkah keamanan standar industri. Namun tidak ada sistem yang 100% aman — gunakan situs ini dengan bijak.

6. Kontak

Untuk pertanyaan seputar privasi, hubungi kami melalui menu Hubungi Kami.

📄 Syarat Penggunaan

1. Layanan

Cinevu menyediakan akses ke konten streaming film dan series secara gratis. Layanan dapat berubah sewaktu-waktu tanpa pemberitahuan sebelumnya.

2. Penggunaan yang Diizinkan

3. Larangan

5. Penafian

Cinevu disediakan "apa adanya" tanpa jaminan apapun. Penggunaan situs sepenuhnya atas risiko pengguna.

❓ FAQ — Pertanyaan Umum

Apakah Cinevu gratis?

Ya! Cinevu sepenuhnya gratis untuk ditonton. Tidak perlu daftar atau bayar apapun.

Kenapa video tidak bisa diputar?

Coba ganti server video dengan menekan tombol server lain di bawah player. Jika semua server error, konten mungkin sedang tidak tersedia sementara.

Apakah ada aplikasi Android/iOS?

Saat ini Cinevu tersedia di browser mobile. Kamu bisa install sebagai PWA (Progressive Web App) dengan menekan "Tambah ke Layar Utama" di browser.

Kualitas video yang tersedia apa saja?

Tergantung server — tersedia mulai dari 360p hingga 1080p (Full HD), tergantung ketersediaan konten.

Bagaimana cara request film?

Hubungi kami melalui menu Hubungi Kami dengan menyebutkan judul film yang diinginkan.

Ada iklan, apakah bisa dihilangkan?

Iklan membantu Cinevu tetap gratis. Kamu bisa menggunakan adblocker, namun beberapa fitur mungkin terpengaruh.

▶️ Cara Menonton di Cinevu

Langkah 1 — Cari Film

Gunakan kolom pencarian di navbar atau jelajahi kategori/genre di halaman utama.

Langkah 2 — Pilih Film

Klik poster film yang ingin kamu tonton. Kamu akan diarahkan ke halaman detail film.

Langkah 3 — Klik Tonton

Tekan tombol ▶ Tonton atau klik langsung pada area player untuk memulai streaming.

Langkah 4 — Pilih Server

Jika video tidak muncul atau buffering lama, coba pilih server lain yang tersedia di bawah player.

Langkah 5 — Nikmati! 🎉

Selamat menonton! Gunakan mode fullscreen untuk pengalaman terbaik.

Tips Tambahan

  • Gunakan WiFi atau data 4G/5G untuk kualitas terbaik
  • Chrome & Firefox direkomendasikan untuk kompatibilitas maksimal
  • Aktifkan mode landscape di HP untuk tampilan lebih lebar
